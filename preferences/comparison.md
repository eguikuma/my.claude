# [ふたつの違い](#comparison) {#comparison}

Claude.ai と Claude Code、どちらを使うべき？

---

## [機能対応表](#feature-comparison) {#feature-comparison}

{: .labeled}

| やりたいこと             | Claude.ai                   | Claude Code                 |
| ------------------------ | --------------------------- | --------------------------- |
| 組織全体で統一したい     | Enterprise Memory Controls  | Enterprise Policy           |
| チームで共有したい       | Project Instructions        | Project Memory（CLAUDE.md） |
| プロジェクト固有の学習   | Project Memory And Summary  | Project Rules               |
| 個人設定（全体）         | Profile Preferences + Style | User Memory                 |
| 個人設定（プロジェクト） | ー                          | Project Memory Local        |
| 一時的な会話             | Incognito Chats             | ー                          |

---

## [主な違い](#main-differences) {#main-differences}

### [Claude.ai](#claude-ai) {#claude-ai}

- ブラウザで使う
- GUIで設定できる
- 会話から自動で学習する
- プロジェクト機能あり

### [Claude Code](#claude-code) {#claude-code}

- ターミナルで使う
- ファイルで設定する（`CLAUDE.md`）
- Gitで設定を共有できる
- コード作業に特化

---

{: .labeled}

| シーン                   | おすすめ               |

| アイデアを相談する       | Claude.ai              |
| コードを書く             | Claude Code            |

| チームで設定を共有したい | Claude Code（Git経由） |
